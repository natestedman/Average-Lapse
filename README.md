AverageLapse
============

This program outputs incremental averages of image sequences (starting from
either the frames of a video file or a sequence of image files).

Build Dependencies
------------------

* BWToolkit (for now, must be installed to ~/Library/Frameworks)
* XCode
* Mac OS X 10.6+
